Adventure Friends

A proof of concept version of this app was my final project for the General Assembly Android Development Immersive program. It is intended to be a fully electronic mobile take on a tabletop role-playing game. Users will be able to play together with one Game Master and several players to make their way through an adventure.

Planned features include several playable races and classes, an alternate map showing hidden items for the game master, player stat sheets and inventory, and built-in game logs. A "reach" feature for this project is an additional device that holds a centralized map for the players, which may be another phone or tablet, or ideally a ChromeCast.
